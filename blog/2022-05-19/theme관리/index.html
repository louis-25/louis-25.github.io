간편하게 테마 스타일링 하는법
CSS / Recoil
2022.05.19.

웹페이지를 스타일링 하다보면

여러가지 테마를 적용하고 싶을때가 있다

예를들어 Dark모드와 Light모드가 있다

이럴때 styled-components를 사용하면 간편하게 스타일링이 가능하다

해당 프로젝트는 TypeScript를 사용하므로 아래와같이 프로젝트를 생성하여 적용해보자

yarn create react-app theme-management --template typescript

1. theme.ts생성

src폴더 아래에 theme.ts 생성

전역적으로 가져다쓸 테마의 핵심이되는 스타일을 정의해주는 부분이다

/* theme.ts */
import { DefaultTheme } from "styled-components";

export const darkTheme: DefaultTheme = {
  bgColor: "black",
  textColor: "white",
};

export const lightTheme: DefaultTheme = {
  bgColor: "white",
  textColor: "black",
};

2. styled.d.ts 생성

src폴더 아래에 styled.d.ts생성

theme.ts에서 모든 테마에서 하나도 빠짐없이 구현해야하는 모든 스타일값을 정의해주는

DefaultTheme 인터페이스를 styled.d.ts에 적어주자

dark테마와 light테마 모두 DefaultTheme 인터페이스를 상속시켜주자

/* styled.d.ts */
import "styled-components";

declare module "styled-components" {
  export interface DefaultTheme {
    textColor: string;
    bgColor: string;
  }
}

3. ThemeProvider로 감싸주기

전역적으로 사용될 컴포넌트들의 부모에

styled-components에서 제공하는 ThemeProvider로 한번 감싸주면

하위 컴포넌트에서 언제든지 theme의 스타일에 접근이 가능하다

/* App.tsx */
import React from "react";
import ReactDOM from "react-dom/client";
import App from "./App";

const root = ReactDOM.createRoot(
  document.getElementById("root") as HTMLElement
);

root.render(
  <React.StrictMode>
    <App />
  </React.StrictMode>
);
/* App.tsx */
import React from "react";
import ReactDOM from "react-dom/client";
import { ThemeProvider } from "styled-components";
import { darkTheme, lightTheme } from "./theme";
import App from "./App";

const root = ReactDOM.createRoot(
  document.getElementById("root") as HTMLElement
);

root.render(
  <React.StrictMode>
    <ThemeProvider theme={darkTheme}>
      <Test />
    </ThemeProvider>
  </React.StrictMode>
);

3-1. Recoil을 사용한 간편한 테마전환

테마를 어느 컴포넌트에서나 접근하여 변경하려면

Recoil과 같은 상태관리 라이브러리를 사용하는것이 편리하다

/* App.tsx */
import React from "react";
import ReactDOM from "react-dom/client";
import { ThemeProvider } from "styled-components";
import { darkTheme, lightTheme } from "./theme";
import { useRecoilValue } from "recoil";
import { isDarkAtom } from "./atoms";
import Test from "./Test";

function App(){
  const isDark = useRecoilValue(isDarkAtom);
  return (
    <>
      <ThemeProvider theme={isDark ? darkTheme : lightTheme}>
        <Test />
      </ThemeProvider>
    </>
  );
}

/* atoms.ts */
import { atom } from "recoil";

export const isDarkAtom = atom({
  key: "isDark",
  default: true,
});

4. 테마 스타일 적용하기

ThemeProvider로 감싸줬으면 하위 컴포넌트에서

스타일을 가져다 사용해보자

styled-components의 createGlobalStyle에 스타일을 적용하면

해당 컴포넌트 및 하위컴포넌트에 전역적으로 스타일링이 적용된다

/* Test.tsx */
import React, { useState } from "react";
import { createGlobalStyle } from "styled-components";

const GlobalStyle = createGlobalStyle`
  body {
    background-color: ${(props) => props.theme.bgColor};
    color: ${(props) => props.theme.textColor};
  }
`;

function Test() {
  return (
    <>
       <GlobalStyle></GlobalStyle>
    </>
  );
}

export default App;
